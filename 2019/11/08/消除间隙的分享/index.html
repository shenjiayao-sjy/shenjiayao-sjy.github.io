消除间隙的分享

Posted on 2019-11-08

问题：采用inline-block布局时会出现div之间有间隙的情况

如以下图片所示

这是它的css

.search{
    width: 400px;
    height: 80px;
    display: inline-block;
    vertical-align: bottom;
}
.searchborder{
    border: solid 3px #F44336;
    height: 35px;
    width: 250px;
    vertical-align: bottom;
}
.searchlogo{
    height: 35px;
    vertical-align: bottom;
}
.inline-block{
    display: inline-block;
}

它的html

<div class="search">
    <div class="inline-block">
        <input type="search" class="searchborder">
    </div>
    <div class="inline-block">
        <img src="photo/search.png" class="searchlogo">
    </div>
</div>

消除间隙的方法

  1. font-size法
  2. 移除空格法
  3. margin负值法
  4. 无闭合标签法
  5. letter-spacing法
  6. Word-spacing法

font-size法

我们可以通过对父元素使用 font-size:0 实现消除间隙。
如以下代码所示

.search{
    width: 400px;
    height: 80px;
    display: inline-block;
    vertical-align: bottom;
    font-size: 0; 
}

效果图为

移除空格法

元素间留白间距出现的原因就是标签段之间的空格，因此，去掉HTML中的空格，自然间距就没了。考虑到代码可读性，显然连成一行的写法是不可取的，我们可以这么写：

<div class="search">
    <div class="inline-block">
        <input type="search" class="searchborder"></div><div class="inline-block">
        <img src="photo/search.png" class="searchlogo"></div>
</div>

或者这么写：

<div class="search">
    <div class="inline-block">
        <input type="search" class="searchborder"></div
        ><div class="inline-block"><img src="photo/search.png" class="searchlogo"></div>
</div>

或者借助HTML注释：

<div class="search">
    <div class="inline-block">
        <input type="search" class="searchborder">
    </div><!--
    --><div class="inline-block">
        <img src="photo/search.png" class="searchlogo">
    </div>
</div>

等。可实现理想效果。

margin负值法

我们可以通过修改元素的左右margin，使其成为负值，从而实现间隙的消除，如下：

.inline-block{
    display: inline-block;
    margin-right: -5px;
}

注意：margin负值的大小并不确定，与上下文的字体和文字大小相关，需自己调试到理想效果。
由于外部环境的不确定性，以及最后一个元素多出的父margin值等问题，这个方法不适合大规模使用。

无闭合标签法

我们可以通过去除子元素的闭合标签来实现效果，由于我的例子并非单纯的div，img与input无闭合标签，下面采用其他例子：

<div class="space">
    <a href="##">惆怅
    <a href="##">淡定
    <a href="##">热血</a>
</div>

考虑到IE6/IE7浏览器的兼容性，最后一个列表的标签的结束（闭合）标签不能丢。
在HTML5中，我们可以直接：

<div class="space">
    <a href="##">惆怅
    <a href="##">淡定
    <a href="##">热血
</div>

letter-spacing法

我们可以通过给父元素设置letter-spacing的负值从而来消除间隙。
如以下代码所示：

.search {
    width: 400px;
    height: 80px;
    display: inline-block;
    vertical-align: bottom;
    letter-spacing: -4px;
}

注意：此方法与margin负值法一样需根据实际情况调整负值的大小。负值小到理想值时再变小也不会有变化。

Word-spacing法

我们可以通过给父元素设置Word-spacing的负值来消除间隙。
如以下代码所示：

.search {
    width: 400px;
    display: inline-block;
    height: 80px;
    vertical-align: bottom;
    word-spacing: -4px;
}

注意：此方法与上一种方法也大同小异，也需自行调整。负值小到理想值时再变小也不会有变化。

总结

目前我只找到了以上几种方法，仍待补充。
总的来说，移除空格和无闭合标签法较不为推荐，改变格式后容易出错，其它几种都比较方便。
参考文献 《去除inline-block元素间间距的N种方法》By赵鑫旭
