js自测题的作答与分析

1.

var a = function(){}
a.b = 1
a.prototype.b = 2
a.prototype.c = 3
a.prototype.d = 4
console.log(a.b) 
console.log(new a().b)

var foo = new a()
foo.c = 5
console.log(foo.c) 
console.log(foo.d)

答：console.log的输出分别为1、2、5、4。

2.

var Foo = function(){
    this.a = 1
    return {
        a:2
    }
}

var bar = new Foo()
console.log(bar.a)

答:console.log的输出为2。

原因为变量bar调用的函数foo所返回的属性a的值为2，即bar所接受的也就a:2。
